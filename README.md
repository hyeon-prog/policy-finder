# 정책파인더 (Policy Finder)

나이·거주지·가구원 수·취업 상태·혼인/자녀 상태 5가지 정보를 입력하면
**정부정책 · 지자체 프로그램 · 공모전**을 유형별 탭으로 나눠 맞춤 추천해 주는 웹 서비스입니다.

🔗 **바로 사용하기: https://hyeon-prog.github.io/policy-finder/**

## 주요 기능

- **맞춤 매칭**: 각 항목이 실제로 요구하는 조건만 검사해 충족률을 계산
  - 모든 조건 충족 + 접수 중 → ✅ 바로 신청 가능
  - 모든 조건 충족 + 접수 마감 → ⏸ 다음 공고 대기
  - 60% 이상 충족 → 💡 관심 항목 (미충족 조건을 배지로 표시)
- **실데이터**: 2026-07-10 기준 각 기관 공고를 확인해 작성한 실제 정책 정보
  (청년월세지원 상시 전환, 국취제 수당 월 60만 원 인상, 청년미래적금 출시 등 반영)
- **유형별 탭**: 전체 / 정부정책 / 지자체 프로그램 / 공모전·대외활동 (건수 배지 포함)
- **웹 접근성**: 스크린리더 대응(aria-live, role=tablist), 키보드 탭 이동, 다크모드 지원
- 입력 정보는 서버로 전송되지 않고 브라우저 안에서만 처리됩니다.

## 구조

```
index.html      # 입력 폼 + 결과 화면 (SEO 메타 포함)
css/style.css   # 자체 CSS — 디자인 토큰 기반 라이트/다크 테마
js/data.js      # 정책·프로그램·공모전 데이터 (DATA_VERIFIED_AT 기준일 명시)
js/app.js       # 매칭 알고리즘 + 탭 + 접수상태 + 렌더링
```

## 데이터 갱신 / 실서비스 확장

- 정책 정보는 `js/data.js`에서 관리합니다. 항목의 `open` 필드로 접수 여부를,
  `DATA_VERIFIED_AT`으로 기준일을 갱신하세요.
- 라이브 데이터 연동 시: [온통청년 청년정책 Open API](https://www.youthcenter.go.kr)
  (공공데이터포털 인증키 필요) + 서버리스 프록시(CORS/키 보호)로 `data.js`를 대체하면 됩니다.
- 실제 자격·기한·금액은 반드시 각 기관의 공식 공고를 확인하세요.
