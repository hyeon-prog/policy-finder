# 정책파인더 (Policy Finder)

나이·거주지·가구원 수·취업 상태·혼인/자녀 상태 5가지 정보를 입력하면
**정부정책 · 지자체 프로그램 · 공모전**을 유형별 탭으로 나눠 맞춤 추천해 주는 웹 프로토타입입니다.

🔗 **바로 사용하기: https://hyeon-prog.github.io/policy-finder/**

## 주요 기능

- **맞춤 매칭**: 각 항목이 실제로 요구하는 조건만 검사해 충족률을 계산
  - 모든 조건 충족 → ✅ 바로 신청 가능
  - 60% 이상 충족 → 💡 관심 항목 (미충족 조건을 배지로 표시)
- **유형별 탭**: 전체 / 정부정책 / 지자체 프로그램 / 공모전·대외활동 (건수 배지 포함)
- **웹 접근성**: 스크린리더 대응(aria-live, role=tablist), 키보드 탭 이동 지원
- 입력 정보는 서버로 전송되지 않고 브라우저 안에서만 처리됩니다.

## 구조

```
index.html      # 입력 폼 + 결과 화면 (Tailwind CDN)
js/data.js      # Mock 데이터 (정책·프로그램·공모전 12건) + 지역 데이터
js/app.js       # 매칭 알고리즘 + 탭 + 렌더링
```

정책 정보는 예시(Mock Data)이며, 실제 신청 조건은 각 기관 공고를 확인하세요.
실서비스 전환 시 `js/data.js`를 API 응답으로 교체하면 됩니다.
